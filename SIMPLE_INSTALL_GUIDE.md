## 🚀 **방법 2: APK 빌드 (네이티브 앱)**

### 클라우드에서 APK 생성

```bash
# 1. EAS CLI 사용
cd /home/user/webapp
npx eas-cli login

# 2. APK 빌드
npx eas-cli build --platform android --profile preview

# 3. 빌드 완료 후 다운로드 링크 받음
# → 스마트폰에서 APK 다운로드 & 설치
```

**빌드 시간:** 10-20분  
**결과:** 직접 설치 가능한 APK 파일

## 🎯 **추천 방법: PWA 설치!**

### 왜 PWA가 좋은가?

1. **즉시 사용 가능** - 설치 파일 불필요
2. **자동 업데이트** - 항상 최신 버전
3. **오프라인 지원** - 인터넷 없이도 작동
4. **용량 작음** - 몇 MB만 사용
5. **앱처럼 작동** - 전체 화면, 홈 아이콘

### PWA 설치 단계 (상세)

#### Android (Chrome)
1. Chrome으로 링크 열기
2. 주소창 오른쪽 **⋮** 탭
3. **"홈 화면에 추가"** 선택
4. **"추가"** 버튼 탭
5. 홈 화면에서 앱 아이콘 확인! 🎉

#### iPhone (Safari)
1. Safari로 링크 열기
2. 하단 **공유 버튼** (□↑) 탭
3. 아래로 스크롤
4. **"홈 화면에 추가"** 선택
5. **"추가"** 버튼 탭
6. 홈 화면에서 앱 아이콘 확인! 🎉

## 💡 **PWA vs 네이티브 앱**

### PWA로 할 수 있는 것:
✅ 모든 화면 사용  
✅ 운동 목록 보기  
✅ 운동 상세 정보  
✅ 식단 추천  
✅ 다크 모드  
✅ 오프라인 모드  
✅ 홈 화면 아이콘

### 네이티브 앱만 가능한 것:
⚠️ 일부 고급 애니메이션  
⚠️ 백그라운드 알림  
⚠️ 일부 센서 접근

**결론:** 대부분의 기능은 PWA로 충분합니다! ✨

### "홈 화면에 추가" 버튼이 안 보여요
